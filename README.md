## Como Jogar

- **Jogador 1 (esquerda)**
  - Mover para cima: `W`
  - Mover para baixo: `S`

- **Jogador 2 (direita)**
  - Mover para cima: `Seta para cima`
  - Mover para baixo: `Seta para baixo`

- Reiniciar partida: `R`
- Sair do jogo: `Z`
